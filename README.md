# CPA Royale — Atualizações

Canal oficial de distribuição do **CPA Royale para Windows**.

Este repositório público contém somente os binários necessários para atualização automática. O código-fonte e o histórico de desenvolvimento permanecem privados em `GustavoCintra97/CPA-Royale`.

[Baixar a versão mais recente](https://github.com/GustavoCintra97/CPA-Royale-Updates/releases/latest)

Arquivos publicados em cada versão:

- `CPA-Royale-Setup-x.y.z.exe`: instalador Windows x64;
- `CPA-Royale-Setup-x.y.z.exe.blockmap`: atualização incremental;
- `latest.yml`: versão, tamanho e hash SHA-512 verificados pelo atualizador.

Os dados operacionais do usuário não são enviados a este repositório.
